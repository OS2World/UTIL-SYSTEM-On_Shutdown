
                              On Shutdown

    On Shutdown is an OS/2 PM program that runs a process during OS/2 
    shutdown.

    Usage:
        onshutdown <program> [arguments]

    Example:
        onshutdown c:\shutdown.cmd

    To install, make a program reference and add it to the startup folder
    or START this program from startup.cmd.

    Feel free to send comments, bug reports, or suggestions to me.
